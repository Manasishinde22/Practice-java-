# Practice-java-

This repository contains **Java practice problems** focused on  
**Data Structures, Algorithms, and Backend-style problem solving**.

It is designed for:
- Product-based company interviews
- Java Backend Developer roles
- DSA + System-thinking preparation

---

## 📂 Topics Covered

### 🔹 Graph Algorithms
- Topological Sort
- Cycle Detection (Directed Graph)
- BFS / DFS
- DP on DAG
- Largest Color Value in Graph

### 🔹 Dynamic Programming
- 0/1 Knapsack
- Interval DP (Minimum Cost to Cut a Stick)
- Bitmask DP (Assignment Problem)
- Memoization & Tabulation

### 🔹 Backend-Oriented Logic
- HashMap-based aggregation (SQL GROUP BY logic)
- Top-K Frequent Elements (Heap + Map)
- Task Scheduling using PriorityQueue
- API-style frequency and analytics problems
